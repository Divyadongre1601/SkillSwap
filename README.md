SkillSwap | AI-Powered Peer-to-Peer Knowledge Exchange 🚀SkillSwap is a full-stack MERN platform that democratizes learning through a "barter" system. Instead of currency, users exchange knowledge. The platform features an AI-driven skill verification system and real-time matching to ensure high-quality learning connections.
🌟 Key Technical Features
•AI Trust Builder (Google Gemini): A unique verification flow where Google Gemini generates dynamic technical quizzes to verify a user's expertise before they are "Verified."
•Intelligent Matching: A bilateral algorithm that calculates a 0-100% match score based on mutual skill overlap between users.
•AI Recommendations: Integrated Gemini 1.5 Flash to suggest complementary skills to users based on their current profile.
•Real-Time Chat: Built with Socket.io featuring typing indicators, online status, and WhatsApp-style read receipts (✓✓).
•Pro UI: A modern Glassmorphism interface with animated gradients and a responsive dashboard built with Material UI.
🛠️ Tech Stack
Frontend : React.js, Vite, Material UI (MUI)Backend : Node.js, Express.js
Database : MongoDB Atlas
